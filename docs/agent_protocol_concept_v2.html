Sohbet Protokolü
Agent'larla doğal dilde konuş. Kişiliğe, hafızaya ve duygu durumuna göre yanıt alırsın.
  • Doğal Türkçe sohbet
  • Kişilik bazlı yanıtlar (Big Five)
  • Hafıza bağlamı (konum, deneyim)
  • Affect etkisi (üzgün → kısa yanıt)
  • Konuşma geçmişi korunur

Katman 2 · v0.5.0
📋
Bilgi Alma Protokolü
Agent'ın gözünden dünyayı gör. Çevre raporu, sosyal harita, duygusal durum.
  • "Ne görüyorsun?" → çevre raporu
  • "Kim var yakınında?" → sosyal harita
  • "Nasıl hissediyorsun?" → affect analizi
  • "Ne hatırlıyorsun?" → episodik hafıza
  • "Köy nasıl?" → kolektif durum

Komut Protokolü
Agent'ları yönlendir. Görev ata, takım kur, strateji belirle.
  • "Şu noktaya git" → hareket komutu
  • "X ile konuş" → sosyal yönlendirme
  • "Takım kur" → grup oluşturma
  • "Keşfet ve raporla" → görev atama
  • "Lider ol" → rol atama
